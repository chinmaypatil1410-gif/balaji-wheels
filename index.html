Firebase Firestore Rules — Action Required
If vehicle / tyre data is not saving, your Firestore Security Rules are blocking writes.
Go to Firebase Console → Firestore → Rules and paste the rules below, then click Publish: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
⚠️ These rules allow any logged-in user to read/write. Suitable for your team setup.
